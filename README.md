# 🤖 Automação de Relatório de Vendas com Python
📌 Descrição do Projeto  Este projeto automatiza o processo de coleta e análise de dados de vendas utilizando Python. O script acessa um arquivo de vendas armazenado no Google Drive, lê os dados em Excel e calcula métricas importantes como faturamento total e quantidade total de produtos vendidos.  Além disso, o processo inclui automação de navegação e abertura de e-mail para envio do relatório.  🎯 Objetivo da Análise  Automatizar a geração de um relatório simples de vendas, reduzindo tarefas manuais e agilizando o acesso a métricas importantes do negócio.  🧹 Tratamento e Limpeza de Dados Leitura do arquivo Excel com Pandas Estruturação dos dados em um DataFrame Utilização de colunas numéricas para cálculo das métricas de vendas 📊 Análises Realizadas Cálculo do faturamento total a partir da soma da coluna Valor Final Cálculo da quantidade total vendida utilizando a coluna Quantidade 🛠️ Tecnologias Utilizadas Python Pandas PyAutoGUI Pyperclip Excel 💡 Principais Insights  O projeto demonstra como Python pode ser utilizado para automatizar tarefas operacionais e gerar relatórios rapidamente, integrando análise de dados com automação de processos.
